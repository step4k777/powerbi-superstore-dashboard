# Superstore Sales Dashboard (Power BI)

## Описание проекта

В рамках проекта был разработан аналитический дашборд в Power BI для анализа продаж на основе датасета **Superstore**.

Дашборд позволяет анализировать ключевые показатели бизнеса, динамику продаж, прибыльность категорий товаров и географическое распределение продаж.

Проект демонстрирует навыки работы с **Power BI, Power Query и визуализацией данных**.

---

## Основные метрики дашборда

В дашборде представлены ключевые KPI:

- Общая прибыль (Total Profit)
- Общий объем продаж (Total Sales)
- Количество проданных товаров (Total Quantity)
- Средняя скидка (Average Discount)

---

## Визуализации

Дашборд включает следующие элементы анализа:

- **Динамика продаж по годам** — позволяет отслеживать изменение продаж во времени.
- **Продажи по категориям товаров** — сравнение продаж между категориями.
- **Прибыль по подкатегориям товаров** — выявление наиболее прибыльных и убыточных товаров.
- **Карта продаж по штатам США** — географическое распределение продаж.
- **Фильтры (Region, Segment)** — интерактивный анализ данных.

---

## Основные инсайты

В ходе анализа были выявлены следующие наблюдения:

1. Продажи демонстрируют устойчивый рост с **2014 по 2017 год**.
2. Категория **Technology** показывает наибольший объем продаж.
3. Некоторые подкатегории товаров, например **Tables**, являются убыточными.
4. Продажи распределены неравномерно по регионам.

## Датасет

Использован учебный датасет **Superstore**, содержащий данные о заказах, продажах, прибыли и клиентах.

Данные включают:

- дату заказа
- категорию и подкатегорию товара
- объем продаж
- прибыль
- регион и штат
